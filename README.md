# ThesisExperimentVRMoL

本プロジェクトは、VR上での記憶の宮殿法を実験・実装するための統合環境です。

1. Docs/specs/ のノートを読み込む

3. Cursor で編集・実装
4. コードは `Docs/Scripts/` に保存しノートと同期
