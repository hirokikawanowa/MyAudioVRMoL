# Presence Promotes Performance on a Virtual Spatial Cognition Task  
*Maneuvrier A. et al., Frontiers in Virtual Reality, 2020*

## 🧠 研究背景と目的
- VR は空間認知研究に適しているが、**Presence（臨場感）と成績の因果関係**は未解明。
- また **Cybersickness・ゲーム経験・性別・視野依存性**などのヒューマンファクターの関与も検討対象。

## 🎯 研究目的
- Presence が空間認知タスク成績に正の影響を持つかを検証。
- 上記の要因がこの関係を媒介・調整するかを評価。

## 🧪 実験方法
| 項目 | 内容 |
|------|------|
| 被験者 | 48名（男女各24、平均20歳） |
| VR環境 | Unity + HTC Vive、屋外仮想空間、テレポート移動 |
| 課題 | ルート学習 → 空間認知テスト（写真/動画/再認）→標識なし再走行 |
| 測定 | Presence Questionnaire, SSQ, ゲーム経験, Rod-and-Frame Test |
| 分析 | 回帰、媒介分析、有意水準 0.05 |

## 📈 結果
- **Presence → 成績向上（β=0.39, p=0.009）**
- **Cybersickness → 成績低下（β=-0.40, p=0.024）**
- ゲーム経験は Presence を高め、Cybersickness を低下（p<0.03）
- **媒介効果なし**（Cybersickness や経験は媒介せず）
- **性別 × Cybersickness の交互作用が有意**
- モデルは成績分散の26%を説明

## 🔍 考察
1. **高Presenceは地図形成と想起を支援**し、空間認知成績を向上。
2. Cybersickness は特に女性で悪影響を与え、Presenceとのトレードオフ関係も。
3. ゲーム経験が Presence/Cybersickness 双方に正影響。
4. 性別の影響は直接的ではなく文化的背景による。
5. 理論的には、表象理論とエコロジカルアプローチの両視点から再解釈が可能。

## ⚠ 限界と今後の課題
- 被験者は若年層中心
- Cybersickness の分布が制限された実験設定
- 他のVR課題・高刺激環境での再検証が必要

## 🚀 応用ポイント
- MoL VRの評価時にPresence強化とCybersickness軽減を両立するUI設計が重要。
- ゲーム経験が少ない対象者には操作習熟フェーズの導入が有効。
- 評価系では、PQ・SSQとパフォーマンススコアをセットで記録・解析すべき。
